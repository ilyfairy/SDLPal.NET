DOS仙剑除错补丁：

DOS仙剑的bug比起同类游戏虽然少上许多，但大宇一直没有出补丁解决，本着对仙剑（尤其是DOS仙剑）的热爱，我做了这个补丁，修正了一些问题，但还不是很完善，今后也许会有补充。

使用方法：解压后覆盖原文件（可以做个备份）
新的故事，即可享受所有的修正，如读取以前的存档，将有一些功能不能用。
